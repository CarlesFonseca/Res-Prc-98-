This project is set to study Active Galactic Nuclei (AGN) jet feedback in galaxies. To do so, it must probe Star Formation Rate (obtained form the WISE survey), as well as radio data retrieved from LOFAR LoTSS value-added catalogue. 
Below is a definition of each file: 
- LoTSS LOFAR - April 4: First analysis of LOFAR LoTSS data (from mosaics in data-release ---> fits file)
- 1.2.1.Function Plotting Querying: Notebook to retrieve WISE and LoTSS data   and create preliminary plots of Luminosity vs. SFR.
- 1.3.Simplified-SFG cutoff: Distinguish between galaxy types, and apply an excess radio intensity cutoff (idea from paper: https://ui.adsabs.harvard.edu/abs/2019A%26A...622A..12H/abstract)
- 1.4.Luminsotiy_rho_z-scaling: Analyze redshift of Star Forming Galaxies (SFG) vs. AGN.
